# Code of Conduct

Rufid is a technical project about Android, USB storage, boot media, and source-buildable payloads. Keep discussion focused, respectful, and useful.

## Expected Behavior

- Be clear and concrete when reporting bugs.
- Assume good faith in technical disagreement.
- Keep criticism tied to code, behavior, documentation, or reproducible results.
- Respect that destructive USB write bugs can be stressful for users.
- Do not post private tokens, signing material, personal data, or exploit details in public issues.

## Unacceptable Behavior

- Harassment, threats, slurs, or personal attacks.
- Posting private credentials or security-sensitive details publicly.
- Pressuring maintainers to add ads, telemetry, opaque binaries, or license-unclear payloads.
- Repeatedly derailing issues away from reproducible technical work.

## Enforcement

Maintainers may edit or remove comments, lock threads, or block accounts when behavior makes the project harder or unsafe to maintain.
