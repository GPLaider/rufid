# Rufid Positioning

## One-Line Pitch

Rufid turns an Android phone into a clean, auditable USB image writer.

## Short Pitch

Need boot media and only have your phone? Rufid writes ISO/IMG files, direct image URLs, and a packaged FreeDOS LiteUSB image to USB mass-storage devices through Android OTG, without ads, analytics, accounts, or opaque APK-derived payloads.

## Store-Style Bullets

- Write bootable USB media from Android.
- Use local ISO/IMG files, direct image URLs, or packaged FreeDOS.
- Pick common official ISO sources for Windows and major Linux distributions.
- Inspect, verify, benchmark, back up, and diagnose USB media.
- Review payload status inside the app.
- No ads, no analytics, no billing SDKs, no remote crash upload.
- Built for F-Droid from source with documented payload provenance.

## Communities To Test

- F-Droid community.
- Android power users.
- Linux install and rescue USB communities.
- Windows install media users who need a phone-first fallback.
- Repair kit, homelab, and field support users.

## Suggested Angles

### Only Your Phone

Your laptop is not available, but your phone and an OTG adapter are. Rufid is built for that moment.

### No Nonsense

No ads. No account. No fake cleanup feature. Just a focused USB writer with local status and local errors.

### Clean Payloads

FreeDOS, UEFI:NTFS, wimlib, and 7-Zip-JBinding are staged from documented upstream inputs, not lifted from an opaque Android APK.

### F-Droid First

The F-Droid path is the full app path. It includes the audited payload set instead of shipping a deliberately incomplete build.
